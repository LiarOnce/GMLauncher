# GMLauncher

A Game Launcher Example with GameMaker:Studio

# 这东西有啥用

只是用GMS制作的启动器范例而已

可以在同一目录下运行外部程序，可以自动（手动）清理内存

再改改就可以当启动器使用了

# 截图

![icon](http://pic.yupoo.com/liaronce/FL9VMrwL/pGrAY.png)

# 注意

需要编译导出后才能运行外部程序，否则program_directory的目录会认定为GMS的Runner目录，会出现没有这个文件的错误也就很正常了。
